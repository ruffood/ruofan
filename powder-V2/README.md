## 主要配料

**蛋白质来源**：大豆分离蛋白、浓缩牛奶蛋白、浓缩乳清蛋白、酵母蛋白粉、大米蛋白粉

**脂肪来源**：亚麻籽油微囊粉、葵花籽油微囊粉、鱼油、DHA 藻油

**碳水来源**：酶解燕麦粉、麦芽糊精、抗性糊精、燕麦纤维

**其他**：胡萝卜粉、车前子壳纤维


## 产品特点

- 零乳糖配方
- 不使用人工甜味剂（以糖醇替代白糖）
- 不含合成香精、色素、增味剂和防腐剂
- 强化 EPA/DHA omega 脂肪酸
- 含叶黄素、植物甾醇
